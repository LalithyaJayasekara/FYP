## 📐 Logo Spacing & Safety Zone

### **Minimum Clear Space**
The logo should have a clear space (breathing room) around it:
- **Horizontal :** Logo width ÷ 4 (minimum)
- **Vertical:** Logo height ÷ 4 (minimum)

```
Example: If logo is 200px wide, leave 50px clear space on each side
```

### **Minimum Size**
- **Digital (web/app):** 64×64 pixels (icon), 200×200 pixels (full logo)
- **Print:** 1 inch (25mm) at 300 DPI minimum
- **Favicon:** 32×32 pixels (hard minimum; 64×64 preferred)

### **Do's & Don'ts**
✅ **DO:**
- Scale proportionally (maintain 1:1 or original ratio)
- Use on contrasting backgrounds (dark logo on light, light logo on dark)
- Maintain full color or use single-color versions
- Place on solid backgrounds

❌ **DON'T:**
- Stretch or skew the logo
- Rotate the logo (unless 90°/180°/270° angles)
- Use on busy/patterned backgrounds
- Add filters, shadows, or effects without approval
- Change colors without reason
- Remove elements or simplify arbitrarily

## 🖼️ Logo Concepts Comparison

### **Concept A: Bridge Icon** ⭐ RECOMMENDED
```
Visual:        [Clinical] ═══════ [Workplace]
               (Stethoscope)  Bridge  (Briefcase)

Color:         Teal (left), Gold (right)
Feeling:       Safe, connected, structured
Best For:      Primary branding, headers, formal use
Memory:        Instantly communicates "bridge"
Global Appeal: Very high (simple, universal symbols)
```

### **Concept B: Interconnected Nodes**
```
Visual:        Clinical ─→ Coordination ─→ Workplace
                         ↓ (Outcome/Impact)

Color:         Teal, Dark Gray, Gold, Coral
Feeling:       Dynamic, real-time, data-driven
Best For:      Dashboards, analytics, modern tech feel
Memory:        Shows flow and coordination well
Global Appeal: Moderate (more abstract)
```

### **Concept C: Upward Arrow with Split Base**
```
Visual:        Clinical + Workplace ↓
                    ↑ (Unified Arrow)
                 Growth/Support

Color:         Teal (left), Gold (right), Gradient up
Feeling:       Growth, progress, balance, modern
Best For:      Minimalist design, modern UI, presentations
Memory:        Clear directionality (improvement)
Global Appeal: High (arrow universally understood)
```
